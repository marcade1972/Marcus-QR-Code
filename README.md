I got tired of paying web sites for making QR codes. I also didn't want to buy an application to make them either. 
I created this using C# and compiliing it with Visual Studio Code. Everything used to create this is free. 
I hope you get good use out of this. If you don't care about the code side of this, after you download the zip file, delete everything except the \bin\Debug\net8.0-windows. 
The net8.0.windows is the folder that contains the compiled application and you can run it using the <b>MarcusQRCodeVaultClean.exe</b>
